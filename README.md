# AI-Driven Smart Manufacturing using Configurable Multi-Agent Systems

## Overview

This project explores the integration of Large Language Models (LLMs) with Configurable Multi-Agent Systems (CMAS) to support intelligent decision-making in smart manufacturing environments. The objective is to enable autonomous software agents to collaborate, exchange information, and optimize manufacturing processes using AI.

---

## Objectives

- Develop an AI-driven smart manufacturing framework
- Integrate Large Language Models with Configurable Multi-Agent Systems
- Improve decision-making in manufacturing environments
- Demonstrate intelligent agent collaboration
- Explore Industry 4.0 and Industry 5.0 applications

---

- Python
